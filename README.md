
# RewardApp Pro

Modern Android Reward App:
- AdMob Rewarded Ads
- Firebase Realtime Database
- Firebase Authentication
- Play Integrity ready
- Coin balance system
- Daily reward
- Referral code system
- Anti-cheat system
- Modern Material UI
- Withdraw system structure
- Leaderboard structure
- Cloud Function ready

## Features
- Anti root
- Anti emulator
- Anti spam click
- Anti fake reward
- Session validation
- SHA-256 signature
- Firebase secure structure
- Coin transaction history

## Setup
1. Add google-services.json to app/
2. Replace AdMob IDs
3. Enable Firebase Auth
4. Enable Firebase Database
5. Build APK

- Firebase App Check
